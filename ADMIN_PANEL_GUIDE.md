# راهنمای دسترسی به Admin Panel و Ticket Panel

## 🔐 نحوه دسترسی به Admin Panel

### روش 1: از طریق URL مستقیم
```
http://127.0.0.1:8000/admin/
```
یا
```
http://localhost:8000/admin/
```

### روش 2: از طریق Navbar سایت (برای Staff)
- اگر کاربر staff باشید، در navbar سایت لینک "⚙️ پنل ادمین" نمایش داده می‌شود
- روی آن کلیک کنید

### روش 3: از طریق داشبورد
```
http://127.0.0.1:8000/admin/dashboard/
```

## 🎧 نحوه دسترسی به Ticket Panel (Support Panel)

### روش 1: از طریق URL مستقیم
```
http://127.0.0.1:8000/accounts/support-panel/
```

### روش 2: از طریق Navbar سایت
- اگر کاربر staff باشید یا به عنوان ادمین به تیکتی اختصاص داده شده باشید
- در navbar سایت لینک "🎧 پنل پشتیبانی" نمایش داده می‌شود
- روی آن کلیک کنید

### روش 3: از طریق Admin Panel
- در header بالای admin panel، لینک "پنل پشتیبانی" وجود دارد
- روی آن کلیک کنید

## 📋 دسترسی‌ها

### Admin Panel (`/admin/`)
- **دسترسی**: فقط کاربران با `is_staff=True`
- **قابلیت‌ها**:
  - مدیریت محصولات
  - مدیریت دسته‌بندی‌ها
  - مدیریت سفارشات
  - مدیریت تیکت‌ها
  - مدیریت تنظیمات سایت
  - اختصاص ادمین به تیکت‌ها

### Ticket Panel (`/accounts/support-panel/`)
- **دسترسی**: 
  - کاربران با `is_staff=True` (دسترسی به همه تیکت‌ها)
  - کاربرانی که به عنوان ادمین به تیکتی اختصاص داده شده‌اند (فقط تیکت‌های اختصاص‌یافته)
- **قابلیت‌ها**:
  - مشاهده لیست تیکت‌ها
  - پاسخ به تیکت‌ها
  - فیلتر بر اساس وضعیت (باز، پاسخ داده شده، بسته شده)

## 🔧 نکات مهم

1. **برای دسترسی به Admin Panel**:
   - باید یک superuser یا staff user ایجاد کنید
   - دستور: `python manage.py createsuperuser`

2. **برای اختصاص ادمین به تیکت**:
   - به Admin Panel بروید
   - بخش "تیکت‌های پشتیبانی" را باز کنید
   - تیکت مورد نظر را انتخاب کنید
   - در بخش "ادمین‌های اختصاص‌یافته"، کاربران مورد نظر را انتخاب کنید

3. **اگر در Google به صورت local نمی‌آید**:
   - این طبیعی است! سایت شما روی localhost اجرا می‌شود
   - فقط از طریق `http://127.0.0.1:8000` یا `http://localhost:8000` قابل دسترسی است
   - برای دسترسی از اینترنت، باید سایت را روی سرور deploy کنید

## 🚀 اجرای سایت

```bash
# اجرای سرور
python manage.py runserver

# سپس در مرورگر:
# http://127.0.0.1:8000/admin/  (برای admin panel)
# http://127.0.0.1:8000/accounts/support-panel/  (برای ticket panel)
```

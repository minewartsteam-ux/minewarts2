# 📖 راهنمای کامل دسترسی به Admin Panel و Ticket Panel

## 🔐 نحوه دسترسی به Admin Panel

### روش 1: از طریق URL مستقیم در مرورگر
```
http://127.0.0.1:8000/admin/
```
یا
```
http://localhost:8000/admin/
```

**نکته مهم**: سایت شما روی localhost اجرا می‌شود و فقط از طریق این آدرس‌ها قابل دسترسی است. این طبیعی است که در Google جستجو نشود چون سایت شما روی سرور محلی (localhost) است و فقط از کامپیوتر خودتان قابل دسترسی است.

### روش 2: از طریق Navbar سایت (برای Staff)
1. ابتدا به سایت بروید: `http://127.0.0.1:8000/`
2. با حساب staff لاگین کنید
3. در navbar بالای صفحه، لینک "⚙️ پنل ادمین" را می‌بینید
4. روی آن کلیک کنید

### روش 3: از طریق داشبورد
```
http://127.0.0.1:8000/admin/dashboard/
```

## 🚀 نحوه اجرای سایت

### مرحله 1: اجرای سرور Django
```bash
cd "f:\mc site\minecraft_shop1 - Copy"
python manage.py runserver
```

### مرحله 2: باز کردن در مرورگر
بعد از اجرای سرور، این پیام را می‌بینید:
```
Starting development server at http://127.0.0.1:8000/
```

حالا در مرورگر خود این آدرس‌ها را باز کنید:
- **سایت اصلی**: http://127.0.0.1:8000/
- **Admin Panel**: http://127.0.0.1:8000/admin/
- **Ticket Panel**: http://127.0.0.1:8000/accounts/support-panel/
- **Dashboard**: http://127.0.0.1:8000/admin/dashboard/

## 👤 ایجاد حساب Staff

برای دسترسی به Admin Panel، باید یک superuser ایجاد کنید:

```bash
python manage.py createsuperuser
```

سپس:
- Username: نام کاربری خود را وارد کنید
- Email: ایمیل خود را وارد کنید (اختیاری)
- Password: رمز عبور قوی انتخاب کنید

## 📋 دسترسی‌ها و قابلیت‌ها

### Admin Panel (`/admin/`)
**دسترسی**: فقط کاربران با `is_staff=True`

**قابلیت‌ها**:
- ✅ مدیریت محصولات (افزودن، ویرایش، حذف)
- ✅ مدیریت دسته‌بندی‌ها
- ✅ مدیریت سفارشات
- ✅ مدیریت تیکت‌های پشتیبانی
- ✅ اختصاص ادمین به تیکت‌ها
- ✅ مدیریت تنظیمات سایت (رنگ‌ها، متن‌ها، عکس پس‌زمینه)
- ✅ Actions برای مدیریت دسته‌ای تیکت‌ها (بستن، باز کردن، علامت‌گذاری)

### Ticket Panel (`/accounts/support-panel/`)
**دسترسی**: 
- کاربران با `is_staff=True` (دسترسی به همه تیکت‌ها)
- کاربرانی که به عنوان ادمین به تیکتی اختصاص داده شده‌اند (فقط تیکت‌های اختصاص‌یافته)

**قابلیت‌ها**:
- ✅ مشاهده لیست تیکت‌ها
- ✅ فیلتر بر اساس وضعیت (باز، پاسخ داده شده، بسته شده)
- ✅ پاسخ به تیکت‌ها
- ✅ مشاهده تاریخچه پیام‌ها
- ✅ لینک مستقیم به ویرایش تیکت در Admin Panel (برای staff)

## 🔧 چرا در Google جستجو نمی‌شود؟

**این کاملاً طبیعی است!** 

سایت شما روی **localhost** (127.0.0.1) اجرا می‌شود که یعنی:
- فقط از کامپیوتر خودتان قابل دسترسی است
- در اینترنت قابل دسترسی نیست
- Google نمی‌تواند آن را پیدا کند

**برای دسترسی از اینترنت**:
1. سایت را روی یک سرور (مثل Heroku، DigitalOcean، یا سرور شخصی) deploy کنید
2. یک دامنه خریداری کنید
3. DNS را تنظیم کنید
4. سپس Google می‌تواند سایت شما را index کند

## 📱 Navigation در Admin Panel

در header بالای Admin Panel این لینک‌ها وجود دارند:
- **پنل مدیریت**: بازگشت به صفحه اصلی admin
- **داشبورد**: مشاهده آمار و اطلاعات کلی
- **پنل پشتیبانی**: رفتن به ticket panel
- **سایت**: بازگشت به سایت اصلی

## 🎯 Navigation در Ticket Panel

در header بالای Ticket Panel:
- **پنل ادمین**: رفتن به admin panel (فقط برای staff)
- **بازگشت به پنل**: بازگشت به لیست تیکت‌ها
- **ویرایش**: ویرایش تیکت در admin panel (فقط برای staff)
